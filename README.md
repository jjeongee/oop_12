### 서비스 주요기능 ⭐️
- 유저의 심리상태 개선을 위한 **컨텐츠 추천**
    - 웹툰 또는 노래
- 자신의 감정을 `행복`,`불안`,`슬픔`으로 정의
    - 각 감정 상황에 맞는 컨텐츠 추천
- 오늘의 감정을 정리하는 **감정일기** 서비스 제공
-  **투두리스트**를 통해 오늘 할일을 보면서 무기력 극복
<br>

### 페이지 구성 💡
##### 로그인 및 회원가입 
- 소셜 로그인
- 유저 정보를 DB에 저장하여, 각 유저의 감정 상태를 저장
##### 메인페이지 
- 오늘의 감정상태
- 오늘의 추천 콘텐츠
- 일기 작성 인터페이스
##### 투두리스트 페이지
- 유저가 할일을 직접 입력해서 추가
- 할일 완료상태 표시
- 할일을 목록에서 삭제하기
##### 일기 작성페이지
- 오늘의 감정을 선택
- 유저가 자유롭게 텍스트 형식으로 느낀점을 기술
##### 일기 열람페이지
- 일기의 제목과 작성 날짜를 오름차순 정렬
- 일기 제목을 누르면 수정 페이지로 이동
##### 감정차트 페이지
- 사용자가 선택한 감정 데이터를 월간 차트로 표시
- 선택한 감정의 종류를 집계
##### 콘텐츠 추천 페이지
- 사용자가 선택한 감정을 바탕으로 웹툰 또는 노래를 추천
- Firebase DB에서 미리 정의해둔 컨텐츠들을 불러옴
- 추가기능: 유저의 피드백 반영
